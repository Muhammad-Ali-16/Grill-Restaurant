# Grill Restaurant Template

This is a simple and responsive HTML/CSS template for a grill-style restaurant website. You can use it for a restaurant landing page, cafe website, or small business site.

## Live Demo
https://grillrestaurant-template.netlify.app/

## Features
- Works on desktop, tablet, and mobile  
- Clean and modern design  
- Easy to change images, text, and colors  
- Built with plain HTML and CSS  

## How to Use
1. Copy or download this repository.  
2. Open `index.html` in your browser to see it.  
3. Replace images and text with your own.  
4. Change colors or fonts in `style.css` if needed.  

## Project Files
- `index.html` — main page  
- `style.css` — styling  
- `images/` — all images used in the template
